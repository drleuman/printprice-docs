# PrintPrice Documentation

Welcome to the official documentation for **PrintPrice**.

PrintPrice is a **Production Intelligence Platform for the print industry**.  
It combines:

- Book Pricing Engine (BPE)
- PDF Preflight
- Production Intelligence
- Capability Matching
- Matchmaker Routing

## Main areas

- **Platform** — architecture and system overview
- **Engines** — core engines
- **API** — contracts and integrations
- **Infrastructure** — deployment and operations
- **Pipeline** — end-to-end production flow
- **Engineering** — standards and technical debt
- **Roadmap** — strategic direction
